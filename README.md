# Python-Based-Breast-Cancer-Survival-Study
Survival analysis of SEER breast cancer cohort using Python. Evaluates Estrogen status, Progesterone Status, nodal involvement, and tumor features as predictors of overall survival. Implements Kaplan-Meier curves, log-rank tests, and multivariable Cox proportional hazards regression to quantify risk factors and survival patterns.
